**RoBoost** is an enhanced third-party replacement for the standard Roblox bootstrapper, providing additional useful features and improvements while bringing back powerful FastFlags functionality.

## ✨ Features

- **🚀 FastFlags Support** - Configure Roblox rendering and gameplay settings with FastFlags
- **🎮 Discord Rich Presence** - Let your friends know what you're playing at a glance
- **🎨 Content Modding** - Simple support for modding of content files (death sounds, cursors, etc.)
- **📍 Server Location** - See where your server is geographically located
- **⚙️ Graphics Configuration** - Ability to configure graphics fidelity and UI experience
- **🎯 Easy to Use** - Streamlined interface for quick access to all features

## ❓ FAQ

**Q: Is this malware?**

A: No. The source code here is viewable to all, and it'd be impossible for anyone to slip anything malicious into the downloads without being noticed. Just be sure you're downloading from the [official GitHub repository](https://github.com/Rloxx/RoBoost).

**Q: Can using this get me banned?**

A: No, it shouldn't. RoBoost doesn't interact with the Roblox client in the same way that exploits do. It only modifies local client settings and doesn't modify game memory or introduce unauthorized functionality.

**Q: What's the difference from RoBoost?**

A: RoBoost is an enhanced version of RoBoost that:

- Brings back and improves FastFlags functionality
- Enhances the UI for easier use
- Adds additional features for customization
- Is actively developed and maintained

## 🚀 Installation

### Method 1: Direct Download

1. Download the [latest release](https://github.com/Rloxx/RoBoost/releases/latest) of RoBoost
2. Run the installer
3. Configure your preferences if needed
4. Click Install

> **Requirements:** Windows 10/11 and [.NET 6 Desktop Runtime](https://aka.ms/dotnet-core-applaunch?missing_runtime=true&arch=x64&rid=win11-x64&apphost_version=6.0.16&gui=true)

### Getting Started

Once installed, RoBoost is added to your Start Menu. You can:

- Launch Roblox directly
- Configure FastFlags
- Customize game settings
- Set up Discord Rich Presence

## 🛠️ Building from Source

### Prerequisites

- Visual Studio 2022 or later
- .NET 6 SDK
- Windows 10/11

### Build Steps

```powershell
# Clone the repository
git clone https://github.com/Rloxx/RoBoost.git
cd RoBoost

# Build the solution
dotnet build RoBoost.sln

# Run the application
cd RoBoost\bin\Debug\net6.0-windows
.\RoBoost.exe
```
